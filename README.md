# Business Search Dropdown

## Overview

This project is a **Business Search Dropdown** built using **React**, **TypeScript**, and the **Atomic Design** pattern. It allows users to search for a business name and displays matching results in a dropdown list.

## Features

* Search businesses dynamically
* Filter matching company names
* Reusable components using Atomic Design
* Built with React Functional Components
* TypeScript for type safety
* Emotion Styled Components for styling

## Technologies Used

* React
* TypeScript
* Vite
* Emotion Styled Components
* Atomic Design

## Getting Started

Clone the repository:

```bash
git clone <repository-url>
```

Go to the project folder:

```bash
cd business-search-dropdown
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

The application will run on your local development server. Open the URL shown in the terminal to view it in your browser.
